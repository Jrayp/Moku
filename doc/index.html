M.new.from_tilemap (tilemap_url, layer_name, tile_width, tile_height[, on_new_cell])
Builds and returns a new moku map from a supplied defold tilemap. Moku maps built this way are automatically linked to the passed tilemap and layer, and are ready for auto-tiling.

Parameters:

  • tilemap_url url Tilemap url
  • layer_name string The name of the layer to build from
  • tile_width number Pixel width of your tiles
  • tile_height number Pixel height of your tiles
  • on_new_cell function A function called on cell creation (optional)

Returns:

    A new moku map

Path-finding

Functions related to path-finding
M.path.find (map, start_cell, end_cell, cost_fn[, cost_fn_arg])
Computes, and returns a path from a given start cell, to a given end cell. Takes a cost function for tile weights. Negative weights designate the cell as impassible. See example project or readme for details.

Parameters:

  • map map A moku map
  • start_cell cell Given start cell
  • end_cell cell Given end cell
  • cost_fn function Cost function
  • cost_fn_arg any Added to the cost_fn argument table under "user" (optional)

Returns:

  1. An array of cells, in order from start to end cell. Nil if no path found.
  2. A table of cell costs. If no path is found a string is returned giving a reason.
